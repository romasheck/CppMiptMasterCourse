# 10.05 Results

Run parameters:

- strings: 262144
- string length: 12
- alphabet: `a-z`
- strings are generated randomly and kept unique
- hash value type: 32-bit unsigned integer

Final collision counts:

| Function | Collisions |
| --- | ---: |
| APHash | 3 |
| DEKHash | 5 |
| BKDRHash | 7 |
| DJBHash | 8 |
| SDBMHash | 9 |
| JSHash | 11 |
| RSHash | 13 |
| PJWHash | 146 |
| ELFHash | 146 |

The best function in this deterministic run was `APHash`; the worst functions were `PJWHash` and `ELFHash`.
The plotted curves grow with the number of strings because all functions map a growing set of distinct strings into
a fixed 32-bit hash space, so repeated values become more likely as in the birthday problem.
